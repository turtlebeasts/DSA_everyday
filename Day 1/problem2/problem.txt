Problem: Find the First Non-Repeating Element in an Array

Output: 5

Input:  arr[] = {1, 2, 3, 1, 2, 3}
Output: -1  // no unique element

Input:  arr[] = {7, 3, 5, 4, 5, 3, 4}
Output: 7
